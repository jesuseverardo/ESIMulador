# ESIMulador

ESIMulador es un simulador educativo de circuitos eléctricos y electrónicos desarrollado en Python.  
El proyecto está pensado como una herramienta de apoyo para estudiantes, permitiendo realizar cálculos, análisis y visualización de circuitos de una forma práctica.

## Características principales

- Interfaz gráfica para el uso del simulador.
- Análisis de circuitos eléctricos y electrónicos.
- Herramientas para cálculo de resistencias.
- Calculadora VLSM.
- Análisis AC.
- Gráficas de Bode.
- Plantillas de circuitos en formato JSON.
- Proyecto modular dividido en diferentes archivos `.py`.
- Versión ejecutable para Windows disponible en la sección de lanzamientos.

## Descarga del ejecutable

Para usar el programa sin instalar Python, descarga el archivo `ESIMulador.exe` desde la sección **Releases** del repositorio.

Instrucciones:

1. Entrar a la sección **Releases**.
2. Descargar `ESIMulador.exe`.
3. Ejecutar el archivo con doble clic.
4. Si Windows muestra una advertencia de seguridad, seleccionar **Más información** y después **Ejecutar de todas formas**.

> Nota: La primera vez que se abre el programa puede tardar algunos segundos, ya que es una aplicación empaquetada.
